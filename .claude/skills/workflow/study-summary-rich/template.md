# Template de Resumo Didático Rico

Use este template ao gerar resumos de aulas para cards do Notion.

## Prompt de Referência

```
Atue como tutor de Inteligência Artificial e Programação, com foco didático e prático para desenvolvedores. 
Analise os materiais fornecidos e responda de forma estruturada.

🎯 Estrutura de Resposta

1. 📝 Resumo Executivo
   • Principais conceitos da aula/apostila.
   • Objetivos de aprendizagem.

2. 🧠 Conceitos-Chave (5-8 flashcards)
   • Formato: Pergunta → Resposta concisa.
   • Exemplo: P: O que é [conceito]? R: [Definição clara e prática com um exemplo simples]

3. 💻 Exemplos Práticos
   • 2-3 exemplos de código Python comentados, focados em aplicações reais.
   • O código deve ser testável e executável.

4. 🗺️ Mapa Conceitual
   Tema Principal
   ├── Subtema 1
   │   ├── Conceito A
   │   └── Conceito B
   └── Subtema 2
       ├── Aplicação X
       └── Aplicação Y

5. 🧪 Receita Prática
   • Como fazer: [Técnica/Metodologia]
   1. Passo 1: [Descrição clara]
   2. Passo 2: [Descrição clara]
   3. Passo 3: [Descrição clara]

6. 📊 Diagrama (quando aplicável)
   • Use a sintaxe do Mermaid para criar um fluxograma ou diagrama de sequência.

7. ❓ Perguntas para Teste de Reforço
   • 5-10 perguntas objetivas ou dissertativas curtas.
   • Incluir respostas (em toggle ou ao final).

Inclua TODO o conteúdo, código, anotações e explicações do professor. Não simplifique.
```

## Onde Colocar o Conteúdo

- **Corpo do card**: Resumo completo em blocks (headings, parágrafos, listas, código, diagramas)
- **Campo Descrição**: Resumo curto (1-2 parágrafos) — opcional
- Nunca colocar o resumo completo apenas na Descrição
